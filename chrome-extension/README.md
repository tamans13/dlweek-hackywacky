# Brainosaur Tab Tracker

Chrome extension that tracks tab switching and sends events to the Brainosaur web app for focus vs distraction metrics.

## Install

1. Chrome → Extensions → Manage Extensions → Developer mode ON
2. Load unpacked → select this `chrome-extension` folder

## Use

1. Start Brainosaur at http://localhost:5173 (or :3000)
2. Extension runs in the background; no UI
3. Tab switches are sent to the app and logged via the API
